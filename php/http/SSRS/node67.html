Change Group Password

  • Actors: Group Administrator
  • Goals: Administrator will change the password to access the group page
  • Preconditions: Administrator is logged onto to the group's administration page
  • Summary: Related use cases:
  • Steps:
    1. Administrator clicks change group password button
    2. Administrator types in the new password twice
    3. Administrator selects save and exit
  • Alternatives: Administrator cancels out of use case
  • Postconditions: the group password is changed
