Edit Major

  • Actors: User
  • Goals: user will be able to change personal attributes
  • Preconditions: user is logged onto GUS
  • Summary: Related use cases:
  • Steps:
    1. User clicks MyProfile button
    2. User makes changes to personal fields
    3. User selects save and exit
  • Alternatives: User cancels out of use case by clicking on a different navigational area or clicking Cancel
  • Postconditions: User's profile and personal page is updated, and user is returned to their home page
