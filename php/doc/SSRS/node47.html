Safety, security and privacy requirements

GUS contains the personal information of over 5,000 users security should be integrated into every facet of this program. The privacy criteria for this system must reflect privacy policies that apply to the University of Idaho, and the security criteria for this system must reflect the need to secure over 5,000 users from identity theft and potential defamation of character.
Colby Blair 2011-02-25
