Send Text

  • Actors: User
  • Goals: user will be able to send texts to other GUS users
  • Preconditions: user is logged onto GUS
  • Summary: Related use cases:
  • Steps:
    1. User clicks text button
    2. User types in the recipient
    3. User types the message
    4. User clicks send
    5. User makes any required changes, recipient, spelling (if required) go to step 4
  • Alternatives: User cancels out of use case by clicking on a different navigational area or clicking Cancel
  • Postconditions: text is sent and user is returned to previous page
