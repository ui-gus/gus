Remove Event

  • Actors: Officer
  • Goals: remove an event on the calendar
  • Preconditions: Officer is logged onto to the group's officer page
  • Summary: Related use cases: Add Event, modify
  • Steps:
    1. Officer clicks Events button
    2. Officer clicks Remove Event button
    3. Officer sends notfications and has the site updated as needed
    4. Officer selects save and exit
  • Alternatives: Officer cancels out of use case by clicking on a different navigational area or clicking Cancel
  • Postconditions: Event has been removed from calendar and site
